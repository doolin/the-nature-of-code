Hexagons

The animations are constructed using the p5.js library (reference documents for p5). At the moment, all the HTML, CSS, and JavaScript is hand-rolled. The page is deployed to S3 via the AWS command line tool invoked from a Makefile, and served by CloudFront. S3 and CloudFront are configured with Terraform.
